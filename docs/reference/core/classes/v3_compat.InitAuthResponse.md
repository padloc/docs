### fromRaw

▸ **fromRaw**(`raw`): [`InitAuthResponse`](../v3_compat.InitAuthResponse)

Restores propertiers from a raw object of the same form generated by
[toRaw](v3_compat.InitAuthResponse.md#toraw). The base implementation blindly
copies over values from the raw object via `Object.assign` so subclasses should
explictly process any propertyies that need special treatment.

Also takes are of validation and "upgrading" in case the raw object has an old
version. Use the protected [\_fromRaw](v3_compat.InitAuthResponse.md#_fromraw)
method to implement subclass-specific behavior.
